# driller

Assert details from a path.

## Requirements

Nothing special...

## Usage

`driller <OPTION> <PATH>`

## Options

| Switch     | Description                          |
|------------|--------------------------------------|
| --absolute | Provide the absolute path.           |
| --basename | Provide the filename from a path.    |
| --dirname  | Provide the dirname from a path.     |
| --scm-root | Find the SCM root (git or hg).       |
